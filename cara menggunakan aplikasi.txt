Aplikasi Pengolahan Data Industri Kecil dan Menengah (IKM) Berbasis WEB

2. Untuk menjalan kan aplikasi pertama kali anda buka chrome terlebih dahulu

4. Pada tab baru ketikan link http://localhost/projectikm2020/auth (akan terhubung ke halaman login)
5. jika anda tidak mempunyai akun administrator, anda dapat melakukan registrasi terlebih dahulu.
